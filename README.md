## Descripción
Aplicación de escritorio **WPF en .NET 8** que simula la generación de un cartón de bingo.  
Creada para demostrar:
- Uso de **.NET 8 (LTS)**
- WPF con XAML y C#
- Publicación como **un solo .exe**
- Buenas prácticas (Git, Releases)
